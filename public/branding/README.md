# EXPERT branding assets

Carpeta de recursos de marca para la web EXPERT.

## Estructura recomendada

- `logos/` logotipo principal, versiones horizontal, isotipo y favicon.
- `avatars/` avatar corporativo para perfil, WhatsApp Business y área privada.
- `covers/` portadas y banners para WhatsApp Business, hero web y cabeceras.
- `badges/` distintivos de partners y entidades colaboradoras.
- `mockups/` referencias visuales de la home y estilo web.
- `guidelines/` guía básica de uso de marca.

## Paleta corporativa

- Azul oscuro principal: `#0D1B2A`
- Azul grisáceo: `#23364D`
- Dorado principal: `#D4A017`
- Dorado claro: `#F2C14E`
- Blanco crema: `#F8F6F1`
- Gris medio: `#9CA3AF`

## Tipografía sugerida

- Títulos: Playfair Display, Cinzel o serif equivalente.
- Texto: Inter, Montserrat o sans-serif equivalente.

## Uso web

Guardar aquí los archivos finales exportados en PNG, SVG y WebP según proceda. No subir variantes duplicadas sin uso real, porque los repositorios no son trasteros digitales, aunque GitHub aguante bastante abuso humano.
